# Варіант 16 — Лабораторна робота №2

> Варіант: 16
> Лаба: LR2 — Функції, рядки, масиви, форми

## Завдання

### Завдання 1: Пошук та заміна

- Текст: "На вечірній прогулянці ми побачили красиву зірку що яскраво сяяла на небі"
- Знайти: "і"
- Замінити на: "и"
- Очікуваний результат: "На вечирний прогулянци ми побачили красиву зирку що яскраво сяяла на неби"

### Завдання 2: Сортування міст за довжиною назви

- Міста (через пробіл): "Рені Сарни Харків Чернігів Кропивницький Мелітополь Бердянськ Ужгород"
- Критерій: за довжиною назви (від короткої до довгої), при однаковій довжині — за алфавітом
- Очікуваний результат: "Рені, Сарни, Харків, Ужгород, Чернігів, Бердянськ, Мелітополь, Кропивницький"

### Завдання 3: Імʼя файлу

- Шлях: "/usr/local/bin/backup_script.html"
- Очікуваний результат:
  - Директорія: "/usr/local/bin"
  - Імʼя файлу: "backup_script"
  - Розширення: "html"

### Завдання 4: Різниця дат

- Дата 1: "07-11-2023"
- Дата 2: "20-04-2025"
- Очікуваний результат: 530 днів (75 тижнів і 5 днів)

### Завдання 5: Генератор паролів

- Довжина пароля: 14
- Вимоги: мінімум 1 велика літера, 1 мала літера, 1 цифра, 1 спецсимвол (!@#$%^&*()-_=+)
- Перевірка складності: довжина ≥ 8, наявність великої, малої, цифри, спецсимволу (0-5 балів)
- Режим: згенерувати 3 паролі, обрати найсильніший (з найвищим балом складності)

### Завдання 6: Пошук унікальних елементів

- Масив: [6, 10, 3, 15, 6, 10, 1, 3, 12, 15, 8, 4]
- Очікуваний результат: [1, 12, 8, 4]

### Завдання 7: Генератор імен тварин

- Склади: "пар ло ні гу зе тір ма ков ша бі"
- Кількість імен: 3
- Складів на імʼя: 3

### Завдання 8: Операції з масивами

- createArray(): випадковий масив (довжина 4-8, значення 1-50)
- Операція: обʼєднати два масиви → видалити дублікати → сортувати **за спаданням**

### Завдання 9: Асоціативний масив

- Дані: українські студенти та їх оцінки (1-12)
  ```php
  $students = [
      "Білоус Ганна" => 9,
      "Герасименко Тимур" => 4,
      "Діденко Олена" => 11,
      "Корнієнко Владислав" => 6,
      "Мороз Юрій" => 12,
      "Руденко Поліна" => 2,
      "Тищенко Євген" => 8,
  ];
  ```
- Сортування 1: за іменем (ksort)
- Сортування 2: за оцінкою (asort)

### Завдання 10: Реєстраційна форма

- Логін для прикладу: "ganna_bil16"
- Поля: логін, пароль (×2), стать (radio), місто (select), хобі (checkbox), про себе (textarea), фото (file)
- Валідація: логін не порожній, пароль ≥ 4 символи, паролі збігаються, стать обрана, місто обране
- Збереження в сесії, автозаповнення при поверненні
- Вибір мови (іконки) → GET → cookie (6 місяців)
- Список міст: Київ, Львів, Одеса, Харків, Дніпро, Запоріжжя, Вінниця, Полтава, Чернігів, Тернопіль

### Завдання 11: Калькулятор функцій

- X = 4, Y = 5
- Функції: sin(x), cos(x), tg(x), sin(x)/cos(x), x^y, x!
- Очікувані результати:

| Функція | Вираз | Результат |
|---------|-------|-----------|
| sin | sin(4) | -0.7568 |
| cos | cos(4) | -0.6536 |
| tg | tan(4) | 1.1578 |
| my_tg | sin(4)/cos(4) | 1.1578 |
| Степінь | 4^5 | 1024 |
| Факторіал | 4! | 24 |
