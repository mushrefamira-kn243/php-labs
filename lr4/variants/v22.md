# Варіант 22 — Лабораторна робота №4

> Варіант: 22
> Лаба: LR4 — MVC-додаток на PHP
> Тема сайту: Садовий центр (Garden center)

## Завдання 1: Структура MVC

Створити скелет веб-додатку з MVC на тему **"Садовий центр"**. Сторінки:
- Головна (статична, з описом садового центру: асортимент рослин, сезонні пропозиції, контакти)
- Форма реєстрації клієнта (з валідацією полів)
- Сторінка перегляду GET та POST параметрів
- Єдина точка входу (`index.php`)

Очікувана структура:
```
garden-center/
├── classes/ (Application, Controller, PageController, Request, Router, View, PageView)
├── config/ (init.php)
├── controllers/ (IndexController, RegformController, ReqviewController, SettingsController)
├── views/
│   ├── layout/ (header.php, footer.php)
│   ├── index/ (main.php)
│   ├── regform/ (form.php, done.php)
│   ├── reqview/ (showrequest.php)
│   └── settings/ (color.php, greeting.php)
├── css/ (style.css)
├── index.php
└── .htaccess
```

## Завдання 2: Модулі

### 2.1 Модуль обробки форми

**Тип форми:** T4 — Реєстрація клієнта садового центру з уподобаннями

**Поля форми:**
- Прізвище (text)
- Ім'я (text)
- По-батькові (text)
- Уподобання (checkboxes, множинний вибір):
  - Кімнатні рослини
  - Садові дерева
  - Квіти та клумби
  - Газонна трава
  - Садовий інвентар
  - Добрива та грунт
  - Декоративні кущі

**Правила перевірки:**
- Прізвище, Ім'я, По-батькові — обов'язкові поля. Якщо порожнє — вказати, яке саме поле не заповнено
- Уподобання — максимум **3** обрані. Якщо більше → "Оберіть не більше 3 уподобань"

### 2.2 Модуль сесій

На окремій сторінці — вибір кольору фону:
- Збереження в `$_SESSION['bg_color']`
- Застосування на всіх сторінках сайту

### 2.3 Модуль Cookie

- Ім'я (text) + Стать (radio: ЧОЛ/ЖІН)
- Cookie на 30 днів
- "Вітаємо Вас, пане/пані ІМ'Я"
