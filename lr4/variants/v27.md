# Варіант 27 — Лабораторна робота №4-5

> Варіант: 27
> Лаба: LR4-5 — MVC-додаток на PHP
> Тема сайту: Театр (Theatre)

## Завдання 1: Структура MVC

Створити скелет веб-додатку з MVC на тему **"Театр"**. Сторінки:
- Головна (статична, з описом театру: репертуар, трупа, історія, контакти)
- Форма реєстрації глядача (з валідацією полів)
- Сторінка перегляду GET та POST параметрів
- Єдина точка входу (`index.php`)

Очікувана структура:
```
theatre/
├── classes/ (Application, Controller, PageController, Request, Router, View, PageView)
├── config/ (init.php)
├── controllers/ (IndexController, RegformController, ReqviewController, SettingsController)
├── views/
│   ├── layout/ (header.php, footer.php)
│   ├── index/ (main.php)
│   ├── regform/ (form.php, done.php)
│   ├── reqview/ (showrequest.php)
│   └── settings/ (color.php, greeting.php)
├── css/ (style.css)
├── index.php
└── .htaccess
```

## Завдання 2: Модулі

### 2.1 Модуль обробки форми

**Тип форми:** T3 — Реєстрація глядача театру

**Поля форми:**
- Ім'я (text) — ім'я глядача
- Стать (radio: ЧОЛ / ЖІН)
- Дата народження: ДД (text), ММ (text), РРРР (text) — три окремі поля

**Правила перевірки:**
- Якщо стать = ЧОЛ і вік < 23 роки → "Не можна зареєструватися — вам має бути щонайменше 23 роки"
- Якщо стать = ЖІН і вік < 20 років → "Не можна зареєструватися — вам має бути щонайменше 20 років"
- Вік обчислюється за введеною датою народження та поточною датою
- Всі поля обов'язкові; дата має бути коректною (валідний день, місяць, рік)

### 2.2 Модуль сесій

На окремій сторінці — вибір кольору фону:
- Збереження в `$_SESSION['bg_color']`
- Застосування на всіх сторінках сайту

### 2.3 Модуль Cookie

- Ім'я (text) + Стать (radio: ЧОЛ/ЖІН)
- Cookie на 30 днів
- "Вітаємо Вас, пане/пані ІМ'Я"
