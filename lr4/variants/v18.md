# Варіант 18 — Лабораторна робота №4-5

> Варіант: 18
> Лаба: LR4-5 — MVC-додаток на PHP
> Тема сайту: Барбершоп

## Завдання 1: Структура MVC

Створити скелет веб-додатку з MVC на тему **"Барбершоп"**. Сторінки:
- Головна (статична, з описом барбершопу, переліком послуг та майстрів)
- Форма реєстрації (з валідацією полів)
- Сторінка перегляду GET та POST параметрів
- Єдина точка входу (`index.php`)

Очікувана структура:
```
barbershop/
├── classes/ (Application, Controller, PageController, Request, Router, View, PageView)
├── config/ (init.php)
├── controllers/ (IndexController, RegformController, ReqviewController, SettingsController)
├── views/
│   ├── layout/ (header.php, footer.php)
│   ├── index/ (main.php)
│   ├── regform/ (form.php, done.php)
│   ├── reqview/ (showrequest.php)
│   └── settings/ (color.php, greeting.php)
├── css/ (style.css)
├── index.php
└── .htaccess
```

## Завдання 2: Модулі

### 2.1 Модуль обробки форми

**Тип форми: T6 — Логін + Паролі + Про себе (Sub3)**

Контекст: реєстрація клієнта барбершопу для онлайн-запису на стрижку.

**Поля форми:**
- Логін (text) — нікнейм для системи запису
- Пароль 1 (password) — пароль
- Пароль 2 (password) — підтвердження пароля
- Про себе (textarea) — побажання щодо стрижки, тип волосся

**Правила перевірки:**
- Логін — одне слово без пробілів (інакше помилка "Логін має бути одним словом без пробілів")
- Логін — мінімум 5 символів
- Логін — НЕ повинен містити цифри (інакше помилка "Логін не може містити цифри")
- Пароль 1 та Пароль 2 мають збігатися (інакше помилка "Паролі не збігаються")
- Пароль — мінімум 5 символів (інакше помилка "Пароль занадто короткий, мінімум 5 символів")

### 2.2 Модуль сесій

На окремій сторінці — вибір кольору фону:
- Форма з варіантами кольорів
- Збереження в `$_SESSION['bg_color']`
- Застосування на всіх сторінках сайту

### 2.3 Модуль Cookie

Форма для введення імені та статі:
- Поля: Ім'я (text), Стать (radio: ЧОЛ/ЖІН)
- Збереження в cookie на 30 днів
- На всіх сторінках: "Вітаємо Вас, пане/пані ІМ'Я"
