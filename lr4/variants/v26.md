# Варіант 26 — Лабораторна робота №4-5

> Варіант: 26
> Лаба: LR4-5 — MVC-додаток на PHP
> Тема сайту: Хостел (Hostel)

## Завдання 1: Структура MVC

Створити скелет веб-додатку з MVC на тему **"Хостел"**. Сторінки:
- Головна (статична, з описом хостелу: номери, ціни, зручності, розташування)
- Форма реєстрації гостя (з валідацією полів)
- Сторінка перегляду GET та POST параметрів
- Єдина точка входу (`index.php`)

Очікувана структура:
```
hostel/
├── classes/ (Application, Controller, PageController, Request, Router, View, PageView)
├── config/ (init.php)
├── controllers/ (IndexController, RegformController, ReqviewController, SettingsController)
├── views/
│   ├── layout/ (header.php, footer.php)
│   ├── index/ (main.php)
│   ├── regform/ (form.php, done.php)
│   ├── reqview/ (showrequest.php)
│   └── settings/ (color.php, greeting.php)
├── css/ (style.css)
├── index.php
└── .htaccess
```

## Завдання 2: Модулі

### 2.1 Модуль обробки форми

**Тип форми:** T2 — Реєстрація гостя хостелу

**Поля форми:**
- Логін (text) — ім'я користувача для бронювання
- Пароль 1 (password)
- Пароль 2 (password) — підтвердження паролю
- E-mail (text) — для підтвердження бронювання

**Правила перевірки:**
- Пароль 1 і Пароль 2 мають збігатися. Якщо ні → "Паролі не збігаються"
- Пароль — мінімум 8 символів. Якщо менше → "Пароль має містити щонайменше 8 символів"
- E-mail — формат: символи@хост.зона (наприклад: guest@hostel.com). Якщо невірний → "Невірний формат електронної пошти"

### 2.2 Модуль сесій

На окремій сторінці — вибір кольору фону:
- Збереження в `$_SESSION['bg_color']`
- Застосування на всіх сторінках сайту

### 2.3 Модуль Cookie

- Ім'я (text) + Стать (radio: ЧОЛ/ЖІН)
- Cookie на 30 днів
- "Вітаємо Вас, пане/пані ІМ'Я"
