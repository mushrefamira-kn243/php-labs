# Варіант 16 — Лабораторна робота №4-5

> Варіант: 16
> Лаба: LR4-5 — MVC-додаток на PHP
> Тема сайту: Зоомагазин

## Завдання 1: Структура MVC

Створити скелет веб-додатку з MVC на тему **"Зоомагазин"**. Сторінки:
- Головна (статична, з описом зоомагазину, категорій товарів та послуг для тварин)
- Форма реєстрації (з валідацією полів)
- Сторінка перегляду GET та POST параметрів
- Єдина точка входу (`index.php`)

Очікувана структура:
```
pet-shop/
├── classes/ (Application, Controller, PageController, Request, Router, View, PageView)
├── config/ (init.php)
├── controllers/ (IndexController, RegformController, ReqviewController, SettingsController)
├── views/
│   ├── layout/ (header.php, footer.php)
│   ├── index/ (main.php)
│   ├── regform/ (form.php, done.php)
│   ├── reqview/ (showrequest.php)
│   └── settings/ (color.php, greeting.php)
├── css/ (style.css)
├── index.php
└── .htaccess
```

## Завдання 2: Модулі

### 2.1 Модуль обробки форми

**Тип форми: T4 — ПІБ + Уподобання (Sub2)**

Контекст: реєстрація власника домашньої тварини в програмі лояльності зоомагазину.

**Поля форми:**
- Прізвище (text) — прізвище власника
- Ім'я (text) — ім'я власника
- По-батькові (text) — по-батькові власника
- Уподобання (checkboxes) — категорії товарів, що цікавлять:
  - Корм для собак
  - Корм для котів
  - Акваріумістика
  - Товари для гризунів
  - Ветеринарні препарати
  - Іграшки для тварин
  - Аксесуари та одяг
  - Товари для птахів

**Правила перевірки:**
- Прізвище — не порожнє (інакше помилка "Поле 'Прізвище' є обов'язковим")
- Ім'я — не порожнє (інакше помилка "Поле 'Ім'я' є обов'язковим")
- По-батькові — не порожнє (інакше помилка "Поле 'По-батькові' є обов'язковим")
- Уподобання — максимум 2 вибрані (інакше помилка "Можна обрати не більше 2 уподобань")

### 2.2 Модуль сесій

На окремій сторінці — вибір кольору фону:
- Форма з варіантами кольорів
- Збереження в `$_SESSION['bg_color']`
- Застосування на всіх сторінках сайту

### 2.3 Модуль Cookie

Форма для введення імені та статі:
- Поля: Ім'я (text), Стать (radio: ЧОЛ/ЖІН)
- Збереження в cookie на 30 днів
- На всіх сторінках: "Вітаємо Вас, пане/пані ІМ'Я"
