# Варіант 17 — Лабораторна робота №4-5

> Варіант: 17
> Лаба: LR4-5 — MVC-додаток на PHP
> Тема сайту: Спортивний магазин

## Завдання 1: Структура MVC

Створити скелет веб-додатку з MVC на тему **"Спортивний магазин"**. Сторінки:
- Головна (статична, з описом спортивного магазину, категорій товарів та акцій)
- Форма реєстрації (з валідацією полів)
- Сторінка перегляду GET та POST параметрів
- Єдина точка входу (`index.php`)

Очікувана структура:
```
sports-store/
├── classes/ (Application, Controller, PageController, Request, Router, View, PageView)
├── config/ (init.php)
├── controllers/ (IndexController, RegformController, ReqviewController, SettingsController)
├── views/
│   ├── layout/ (header.php, footer.php)
│   ├── index/ (main.php)
│   ├── regform/ (form.php, done.php)
│   ├── reqview/ (showrequest.php)
│   └── settings/ (color.php, greeting.php)
├── css/ (style.css)
├── index.php
└── .htaccess
```

## Завдання 2: Модулі

### 2.1 Модуль обробки форми

**Тип форми: T5 — Калькулятор (Sub2)**

Контекст: калькулятор вартості спортивного обладнання.

**Поля форми:**
- Число 1 (text) — перше значення (наприклад, ціна товару)
- Число 2 (text) — друге значення (наприклад, кількість одиниць)
- Функція (select) — математична операція:
  - Сума (+)
  - Різниця (−)
  - Множення (×)
  - Ділення (÷)
- Результат (text) — очікуваний результат від користувача

**Правила перевірки:**
- Сума, різниця, множення — будь-які числа
- Ділення — Число 2 ≠ 0 (інакше помилка "Ділення на нуль неможливе")
- Обчислити результат на сервері, порівняти з полем "Результат" → невідповідність = помилка "Результат обчислення не збігається"

### 2.2 Модуль сесій

На окремій сторінці — вибір кольору фону:
- Форма з варіантами кольорів
- Збереження в `$_SESSION['bg_color']`
- Застосування на всіх сторінках сайту

### 2.3 Модуль Cookie

Форма для введення імені та статі:
- Поля: Ім'я (text), Стать (radio: ЧОЛ/ЖІН)
- Збереження в cookie на 30 днів
- На всіх сторінках: "Вітаємо Вас, пане/пані ІМ'Я"
