# Варіант 28 — Лабораторна робота №4-5

> Варіант: 28
> Лаба: LR4-5 — MVC-додаток на PHP
> Тема сайту: Ресторан (Restaurant)

## Завдання 1: Структура MVC

Створити скелет веб-додатку з MVC на тему **"Ресторан"**. Сторінки:
- Головна (статична, з описом ресторану: меню, шеф-кухар, атмосфера, бронювання столиків)
- Форма реєстрації гостя (з валідацією полів)
- Сторінка перегляду GET та POST параметрів
- Єдина точка входу (`index.php`)

Очікувана структура:
```
restaurant/
├── classes/ (Application, Controller, PageController, Request, Router, View, PageView)
├── config/ (init.php)
├── controllers/ (IndexController, RegformController, ReqviewController, SettingsController)
├── views/
│   ├── layout/ (header.php, footer.php)
│   ├── index/ (main.php)
│   ├── regform/ (form.php, done.php)
│   ├── reqview/ (showrequest.php)
│   └── settings/ (color.php, greeting.php)
├── css/ (style.css)
├── index.php
└── .htaccess
```

## Завдання 2: Модулі

### 2.1 Модуль обробки форми

**Тип форми:** T4 — Реєстрація гостя ресторану з кулінарними уподобаннями

**Поля форми:**
- Прізвище (text)
- Ім'я (text)
- По-батькові (text)
- Уподобання (checkboxes, множинний вибір):
  - Українська кухня
  - Італійська кухня
  - Японська кухня
  - Французька кухня
  - Вегетаріанське меню
  - Десерти та випічка
  - Авторські коктейлі

**Правила перевірки:**
- Прізвище, Ім'я, По-батькові — обов'язкові поля. Якщо порожнє — вказати, яке саме поле не заповнено
- Уподобання — максимум **4** обрані. Якщо більше → "Оберіть не більше 4 уподобань"

### 2.2 Модуль сесій

На окремій сторінці — вибір кольору фону:
- Збереження в `$_SESSION['bg_color']`
- Застосування на всіх сторінках сайту

### 2.3 Модуль Cookie

- Ім'я (text) + Стать (radio: ЧОЛ/ЖІН)
- Cookie на 30 днів
- "Вітаємо Вас, пане/пані ІМ'Я"
