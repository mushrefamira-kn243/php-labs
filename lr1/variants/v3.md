# Варіант 3 — Лабораторна робота №1

> Варіант: 3
> Лаба: LR1 — Базові конструкції PHP

## Завдання

### Завдання 1: Форматований текст

Створити index.php, який виводить:

```
    Осінній <b>дощ</b> стукає в шибку,
    Листя кружляє <i>повільно</i> в дворі,
    Каштани впали на стежку,
    Туман лягає на гори.
```

**Вимоги:** жирний шрифт (`<b>`), курсив (`<i>`), відступи (`margin-left`)

### Завдання 2: Конвертер валют

- Сума: 25000 грн
- Курс: 1 долар = 42.10 грн
- Очікуваний результат: "25000 грн. можна обміняти на 593.82 долар"

### Завдання 3: Визначення сезону (if-else)

- Місяць для перевірки: 11
- Очікуваний результат: "осінь"

### Завдання 4: Голосний чи приголосний (switch)

- Символ для перевірки: 'о'
- Очікуваний результат: "голосна"

### Завдання 5: Тризначне число

- Число: 916
- Очікувані результати:
  1. Сума цифр: 16
  2. Зворотне число: 619
  3. Найбільше число: 961

### Завдання 6: Цикли

1. Таблиця: 3 x 6 комірок різного кольору
2. Квадрати: 5 червоних квадратів на чорному тлі
