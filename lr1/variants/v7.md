# Варіант 7 — Лабораторна робота №1

> Варіант: 7
> Лаба: LR1 — Базові конструкції PHP

## Завдання

### Завдання 1: Форматований текст

Створити index.php, який виводить:

```
    На <b>горі</b> стоїть старий дуб,
    Його гілки торкаються <i>хмар</i> високих,
    Він пережив не один сум,
    І бачив днів безліч далеких.
```

**Вимоги:** жирний шрифт (`<b>`), курсив (`<i>`), відступи (`margin-left`)

### Завдання 2: Конвертер валют

- Сума: 780 грн
- Курс: 1 долар = 43.00 грн
- Очікуваний результат: "780 грн. можна обміняти на 18.14 долар"

### Завдання 3: Визначення сезону (if-else)

- Місяць для перевірки: 10
- Очікуваний результат: "осінь"

### Завдання 4: Голосний чи приголосний (switch)

- Символ для перевірки: 'і'
- Очікуваний результат: "голосна"

### Завдання 5: Тризначне число

- Число: 854
- Очікувані результати:
  1. Сума цифр: 17
  2. Зворотне число: 458
  3. Найменше число: 458

### Завдання 6: Цикли

1. Таблиця: 8 x 4 комірок різного кольору
2. Квадрати: 7 червоних квадратів на чорному тлі (по 5 у рядку)

## Примітки

- Дивись demo/ для прикладу реалізації
- Структура файлів має відповідати demo
