# Варіант 2 — Лабораторна робота №1

> Варіант: 2
> Лаба: LR1 — Базові конструкції PHP

## Завдання

### Завдання 1: Форматований текст

Створити index.php, який виводить:

```
    Над полем <b>вітер</b> тихо лине,
    Колоски хитає <i>золотий</i> прибій,
    Хмарки пливуть немов перини,
    А літо дарує спокій свій.
```

**Вимоги:** жирний шрифт (`<b>`), курсив (`<i>`), відступи (`margin-left`)

### Завдання 2: Конвертер валют

- Сума: 8700 грн
- Курс: 1 долар = 39.50 грн
- Очікуваний результат: "8700 грн. можна обміняти на 220.25 долар"

### Завдання 3: Визначення сезону (if-else)

- Місяць для перевірки: 7
- Очікуваний результат: "літо"

### Завдання 4: Голосний чи приголосний (switch)

- Символ для перевірки: 'к'
- Очікуваний результат: "приголосна"

### Завдання 5: Тризначне число

- Число: 583
- Очікувані результати:
  1. Сума цифр: 16
  2. Зворотне число: 385
  3. Найбільше число: 853

### Завдання 6: Цикли

1. Таблиця: 5 x 4 комірок різного кольору
2. Квадрати: 8 червоних квадратів на чорному тлі

## Примітки
