## Завдання

### Завдання 1: Форматований текст

Створити index.php, який виводить:

```
    Котик <b>спить</b> на теплій печі,
    За вікном мете <i>холодна</i> хуртовина,
    Бабуся вяже рукавиці,
    А в хаті пахне калина.
```

**Вимоги:** жирний шрифт (`<b>`), курсив (`<i>`), відступи (`margin-left`)

### Завдання 2: Конвертер валют

- Сума: 6350 грн
- Курс: 1 долар = 38.20 грн
- Очікуваний результат: "6350 грн. можна обміняти на 166.23 долар"

### Завдання 3: Визначення сезону (if-else)

- Місяць для перевірки: 2
- Очікуваний результат: "зима"

### Завдання 4: Голосний чи приголосний (switch)

- Символ для перевірки: 'д'
- Очікуваний результат: "приголосна"

### Завдання 5: Тризначне число

- Число: 639
- Очікувані результати:
  1. Сума цифр: 18
  2. Зворотне число: 936

### Завдання 6: Цикли
