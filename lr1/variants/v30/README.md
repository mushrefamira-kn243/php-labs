# Варіант 30 — Повний приклад виконаної лабораторної

Це **повністю розв'язаний варіант 30** лабораторної роботи №1. Використовуйте як зразок разом з `demo/`.

## Різниця між demo/ та v30/

| | demo/ | v30/ |
|---|-------|------|
| Що це | Інтерактивний приклад від викладача | Повне рішення варіанту 30 |
| Дані | Демонстраційні (не з жодного варіанту) | Точно з v30.md |
| Для чого | Зрозуміти як працює кожне завдання | Побачити як виглядає здана робота |

## Запуск

```bash
php -S localhost:8000
```

Відкрийте: `http://localhost:8000/lr1/variants/v30/`

## Структура файлів

```text
v30/
├── index.php           ← завдання 1: перша програма (вірш)
├── task2.php           ← завдання 2: конвертер валют
├── task3.php           ← завдання 3: сезон за номером місяця
├── task4.php           ← завдання 4: голосний чи приголосний
├── task5.php           ← завдання 5: тризначне число
├── task7_table.php     ← завдання 6.1: HTML-таблиця
├── task7_squares.php   ← завдання 6.2: квадрати
└── layout.php          ← спільний шаблон сторінки
```

## Як використовувати як шаблон

1. Скопіюйте цю папку: `cp -r v30 vN` (де N — ваш номер варіанту)
2. Відкрийте ваш `vN.md` — там ваші конкретні дані (вірш, суми, місяці)
3. Замініть дані в кожному файлі на свої з `vN.md`
4. Перевірте: `http://localhost:8000/lr1/variants/vN/`

> **Не змінюйте цю папку** — вона доступна всім студентам як приклад.
