# Варіант 13 — Лабораторна робота №1

> Варіант: 13
> Лаба: LR1 — Базові конструкції PHP

## Завдання

### Завдання 1: Форматований текст

Створити index.php, який виводить:

```
    Зимовий <b>ліс</b> стоїть у тиші,
    Сніжинки падають <i>повільно</i> вниз,
    Під білим покривалом сплять дерева,
    А вітер наспівує свій капріз.
```

**Вимоги:** жирний шрифт (`<b>`), курсив (`<i>`), відступи (`margin-left`)

### Завдання 2: Конвертер валют

- Сума: 22 доларів
- Курс: 1 долар = 38.25 грн
- Очікуваний результат: "22 доларів можна обміняти на 841.50 грн"

### Завдання 3: Визначення сезону та позиції місяця (if-else)

- Місяць для перевірки: 2
- Очікуваний результат: "зима, останній місяць сезону"

### Завдання 4: Тип символу (switch)

- Символ для перевірки: 'н'
- Категорії: голосна / приголосна / спеціальний символ (ь, ')
- Очікуваний результат: "приголосна"

### Завдання 5: Тризначне число

- Число: 714
- Очікувані результати:
  1. Сума цифр: 12
  2. Зворотне число: 417
  3. Найбільше число: 741

### Завдання 6: Цикли

1. Шахова таблиця: 7 x 8 комірок (чергування чорних і білих)
2. Кола: 5 синіх кіл на білому тлі

## Примітки

- Дивись demo/ для прикладу реалізації
- Структура файлів має відповідати demo
