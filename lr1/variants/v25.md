# Варіант 25 — Лабораторна робота №1

> Варіант: 25
> Лаба: LR1 — Базові конструкції PHP

## Завдання

### Завдання 1: Форматований текст

Створити index.php, який виводить:

```
    Світанок <b>ніжно</b> фарбує небо,
    Роса на листках <i>блищить</i> як кришталь,
    Півень кукурікає голосно й хлібно,
    А ранок прогонить нічну печаль.
```

**Вимоги:** жирний шрифт (`<b>`), курсив (`<i>`), відступи (`margin-left`)

### Завдання 2: Конвертер валют

- Сума: 9500 грн
- Курс: 1 євро = 47.10 грн
- Комісія банку: 1%
- Очікуваний результат: "9500 грн = 201.70 євро, після комісії 1% — 199.68 євро"

### Завдання 3: Визначення сезону та днів у місяці (if-else)

- Місяць для перевірки: 2
- Очікуваний результат: "зима", 28 днів у місяці

### Завдання 4: Латинська літера — голосна чи приголосна (switch)

- Символ для перевірки: 'u'
- Очікуваний результат: "голосна"

### Завдання 5: Тризначне число

- Число: 937
- Очікувані результати:
  1. Сума цифр: 19
  2. Зворотне число: 739
  3. Найменше число: 379

### Завдання 6: Цикли

1. Смугаста таблиця: 7 x 9 комірок (чергування рядків двох кольорів)
2. Трикутники: 22 зелених трикутників на сірому тлі (по 5 у рядку)

## Примітки

- Дивись demo/ для прикладу реалізації
- Структура файлів має відповідати demo
