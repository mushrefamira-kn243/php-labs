# Варіант 23 — Лабораторна робота №1

> Варіант: 23
> Лаба: LR1 — Базові конструкції PHP

## Завдання

### Завдання 1: Форматований текст

Створити index.php, який виводить:

```
    Хвилі <b>моря</b> шумлять на світанку,
    Берег пісний <i>вкриває</i> прибій,
    Чайки летять у ранкову мандрівку,
    А горизонт зливається з мрій.
```

**Вимоги:** жирний шрифт (`<b>`), курсив (`<i>`), відступи (`margin-left`)

### Завдання 2: Конвертер валют

- Сума: 26500 грн
- Курс: 1 євро = 46.20 грн
- Комісія банку: 1.5%
- Очікуваний результат: "26500 грн = 573.59 євро, після комісії 1.5% — 564.99 євро"

### Завдання 3: Визначення сезону та днів у місяці (if-else)

- Місяць для перевірки: 8
- Очікуваний результат: "літо", 31 днів у місяці

### Завдання 4: Латинська літера — голосна чи приголосна (switch)

- Символ для перевірки: 'a'
- Очікуваний результат: "голосна"

### Завдання 5: Тризначне число

- Число: 746
- Очікувані результати:
  1. Сума цифр: 17
  2. Зворотне число: 647
  3. Найбільше число: 764

### Завдання 6: Цикли

1. Смугаста таблиця: 5 x 11 комірок (чергування рядків двох кольорів)
2. Трикутники: 18 зелених трикутників на сірому тлі
