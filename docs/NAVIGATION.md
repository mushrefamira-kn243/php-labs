# Navigation Structure

## Схема

```text
┌─────────────────────────────────────────────────────────────────┐
│                         HOME (/)                                │
│                    [Вибір варіанту: v1 ▼]                       │
└─────────────────────────────┬───────────────────────────────────┘
                              │ Вибрав варіант
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  [← Назад до вибору варіанту]              Варіант N            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ЛР1 — Базові конструкції PHP                                  │
│    [Demo]  [Завдання (vN.md) ↗]  [Опис лаби ↗]                │
│                                                                 │
│  ЛР2 — Масиви та рядки                                         │
│    [Demo]  [Завдання (vN.md) ↗]  [Опис лаби ↗]                │
│  ...                                                            │
└─────────────────────────────────────────────────────────────────┘
          │                              │
          │ Клік "Demo"                  │ Клік "Завдання"
          ▼                              ▼
┌──────────────────────┐    ┌──────────────────────┐
│  Demo Index          │    │  GitHub (new tab)     │
│  [Головна] [← Демо]  │    │  vN.md rendered      │
│  Task cards          │    └──────────────────────┘
└──────┬───────────────┘
       │ Клік на task
       ▼
┌──────────────────────┐
│  Demo Task Page      │
│  [Головна] [← Демо]  │
│  [← Варіант N]*      │
│  Task selector ▼     │
└──────────────────────┘

* "← Варіант N" — тільки LR1 (при переході з ?from=vN)
```

---

## URL Structure

```text
/                                → Home (вибір варіанту)
/?variant=v5                     → Variant page (список лаб для варіанту 5)
/lr1/demo/                       → Demo index (список задач ЛР1)
/lr1/demo/task2.php              → Demo task page
/lr1/demo/task2.php?from=v5      → Demo task page (з контекстом варіанту)
/lr1/variants/v30/               → Reference implementation (варіант 30)
```

> Завдання (vN.md) та опис лаби (assignment.md) відкриваються на GitHub в новій вкладці.

## Pages

### 1. Home (`/`)

Вибір варіанту (1-30) з dropdown.

```text
+--------------------------------------------------+
|                    PHP Labs                       |
|        Серверні технології — 30 варіантів         |
|                                                   |
|          Варіант: [v1 ▼]  [Перейти]              |
+--------------------------------------------------+
```

### 2. Variant Page (`/?variant=v5`)

Список лабораторних з посиланнями на demo, завдання, опис.

```text
+--------------------------------------------------+
|  [← Назад до вибору варіанту]                    |
|                  Варіант 5                        |
+--------------------------------------------------+
|  ЛР1 — Базові конструкції PHP                   |
|    [Demo]  [Завдання (v5.md) ↗]  [Опис лаби ↗]  |
|                                                   |
|  ЛР2 — Масиви та рядки                           |
|    [Demo]  [Завдання (v5.md) ↗]  [Опис лаби ↗]  |
|  ...                                              |
+--------------------------------------------------+
```

### 3. Demo Index (`/lr1/demo/`)

Список задач конкретної лабораторної з карточками.

```text
+--------------------------------------------------+
|  [Головна]                      Демо ЛР1        |
+--------------------------------------------------+
|  Task cards: [2] [3] [4] [5] [6] [7.1] [7.2]   |
+--------------------------------------------------+
```

### 4. Demo Task Page (`/lr1/demo/task2.php`)

Сторінка конкретного завдання з header-навігацією.

```text
+------------------------------------------------------------------+
|  [Головна]  [← Демо]  [← Варіант N]*  |  Демо  |  Task selector ▼ |
+------------------------------------------------------------------+
|                                                                    |
|                        Demo Content                                |
|                                                                    |
+------------------------------------------------------------------+
```

---

## Navigation Buttons

| Button | Description | Location |
| ------ | ----------- | -------- |
| `[Головна]` | На головну `/` | Demo index, Demo task pages |
| `[← Назад до вибору варіанту]` | Скинути варіант | Variant page |
| `[← Демо]` | До demo index | Demo task pages |
| `[← Варіант N]` | До студентського варіанту* | Demo task pages (LR1, при `?from=vN`) |
| `Task selector ▼` | Dropdown вибору задачі | Demo task pages |

## File Structure

```text
php-labs/
├── index.php                     # Home + Variant page
│
├── lr1/
│   ├── demo/
│   │   ├── index.php             # Demo index (task cards)
│   │   ├── layout.php            # Shared layout (header + nav)
│   │   ├── demo.css              # Demo styles
│   │   └── task2.php ... task7_squares.php
│   ├── variants/
│   │   ├── v1.md ... v30.md      # Variant assignments (markdown)
│   │   └── v30/                  # Reference implementation
│   └── assignment.md             # General lab description
│
├── lr2/, lr3/                    # Same pattern (demo/ + variants/ + assignment.md)
├── lr4/                          # MVC demo (different structure)
└── shared/                       # Global CSS, helpers, templates
```

## Student Workflow

1. `/` → вибрати свій варіант (v5)
2. Variant page → побачити список лаб з посиланнями
3. Клік "Demo" → `/lr1/demo/` → вивчити робочий приклад
4. Клік "Завдання (v5.md)" → GitHub (нова вкладка) → прочитати задачі
5. **ЛР1:** скопіювати `lr1/variants/v30/` → `lr1/variants/v5/` → замінити дані
6. **ЛР2-4:** створити файли в `lrN/` за прикладом з demo/
7. `php -S localhost:8000` → перевірити в браузері
8. Закомітити та push на GitHub
