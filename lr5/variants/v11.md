# Варіант 11 — Лабораторна робота №5

> Варіант: 11
> Лаба: LR5 — Файли та база даних (PDO)
> Тема сайту: Тренажерний зал

## Завдання 1: Робота з файлами

### 1.1 Гостьова книга

- Форма: **Ім'я** + **Коментар** → збереження у файл `data/comments.csv`
- Формат: CSV (`дата,ім'я,коментар`)
- Виведення всіх коментарів з файлу в таблицю (з датою)

### 1.2 Завантаження зображень

- Форма upload зображень (JPEG, PNG, GIF, WebP, до 5 МБ)
- Збереження в `data/uploads/`
- Галерея всіх завантажених зображень

### 1.3 Робота з каталогами

- Форма: **Логін** + **Пароль** → створити папку `data/users/{логін}/` з підпапками `video`, `music`, `photo` + по 2-3 файли
- Якщо папка існує → помилка
- Окрема сторінка видалення: **Логін** + **Пароль** → видалити папку з усім вмістом

## Завдання 2: База даних (PDO)

### 2.1 CRUD — Тренування

Таблиця `trainings`:

| Поле | Тип | Опис |
|------|-----|------|
| id | INT (PK, AUTO_INCREMENT) | Унікальний ідентифікатор |
| name | VARCHAR(100) | Назва тренування |
| type | VARCHAR(50) | Тип (силове, кардіо, розтяжка, ...) |
| duration_min | INT | Тривалість (хвилин) |
| trainer | VARCHAR(100) | Ім'я тренера |
| max_participants | INT | Максимум учасників |
| created_at | DATETIME | Дата створення |

**Сторінки:**
- Список всіх записів у таблиці
- Додавання нового запису (форма з валідацією)
- Редагування існуючого запису (форма з передзаповненням)
- Видалення запису за ID

**Вимоги:** prepared statements (`prepare` + `execute`), `try/catch` для PDO

### 2.2 Авторизація через БД

Таблиця `users` (10 полів):

| Поле | Тип |
|------|-----|
| id | INT (PK) |
| login | VARCHAR(50) UNIQUE |
| password | VARCHAR(255) |
| email | VARCHAR(100) |
| first_name | VARCHAR(50) |
| last_name | VARCHAR(50) |
| phone | VARCHAR(20) |
| city | VARCHAR(50) |
| gender | VARCHAR(10) |
| about | TEXT |
| created_at | DATETIME |

**Правила:**
- Унікальний логін
- Пароль: мінімум 6 символів
- E-mail формат
- Хешування: `password_hash` / `password_verify`

**Сторінки:** реєстрація, вхід, профіль, редагування, видалення акаунту, вихід

### 2.3 Сесії та Cookie (з ЛР4)

- Колір фону → `$_SESSION['bg_color']` (5+ кольорів за темою)
- Привітання → cookie на 30 днів

**Кольори за темою (5):**

| Назва | HEX |
|-------|-----|
| Сталевий | `#4a6fa5` |
| Антрацит | `#2d2d2d` |
| Помаранчевий акцент | `#e85d04` |
| Сірий зал | `#6c757d` |
| Білий спорт | `#f8f9fa` |
